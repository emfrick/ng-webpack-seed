### Purpose
A seed project for using Angular 1.x with Webpack.

### Includes
- Bootstrap
- ES6 Support
- Source Maps
- Development Server

### Requirements
- node 6.2.x
- npm 3.9.x

### Installation
```
# Clone the repo
$ git clone <url>

# Move into the directory
$ cd ng-webpack-seed

# Install dependencies
$ npm install

# Run the development Server
$ npm run dev
```
